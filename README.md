# Student Management System (Python Tkinter)

## Overview
This is a **Desktop Student Management System** built using **Python** and **Tkinter GUI**.  
It allows users to **add, view, edit, and delete student records**.  
All data is stored locally in a **CSV file** (`students.csv`).  

This project is beginner-friendly, easy to understand, and demonstrates **Python CRUD operations**, **file handling**, and **GUI development**.

---

## Features
- ✅ Add student with **Name, Email, and Grade**
- ✅ View all students in a **table**
- ✅ Edit student details
- ✅ Delete student
- ✅ Click on a student row to **auto-fill inputs for editing**
- ✅ Data stored in **CSV file** for persistence
- ✅ **Interactive GUI** with buttons and table

---

## Tech Stack
- **Language:** Python 3.x  
- **GUI:** Tkinter  
- **Storage:** CSV file (`students.csv`)
